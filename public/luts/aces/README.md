# ACES LUTs

This directory should contain the ACES (Academy Color Encoding System) Look Up Tables.

## Required Files

For full functionality, place the following .cube files here:

- `ACEScg_to_Rec709.cube` (RRT + ODT for Rec.709)
- `ACEScg_to_sRGB.cube` (RRT + ODT for sRGB)
- `ACEScg_to_P3D65.cube` (RRT + ODT for P3 Display)
- `ACEScg_to_Rec2020.cube` (RRT + ODT for Rec.2020)

## Camera Input Transforms (Optional)

If you want to use LUT-based IDTs instead of the algorithmic ones:

- `SLog3_to_ACEScg.cube`
- `VLog_to_ACEScg.cube`
- `CLog3_to_ACEScg.cube`
- `LogC3_to_ACEScg.cube`

## Sourcing These Files

You can generate these using OpenColorIO (OCIO) or download baked LUTs from the ACES Central repository.

*Current files are placeholders generated by the application.*
